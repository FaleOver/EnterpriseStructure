# Приложение для отображения и взаимодействия со структурой предприятия

Для запуска установите проект **Presentation** в качестве стартового. В приложении настроено **автоматическое применение миграций**. При первом запуске база данных EnterpriseStructureDb и все таблицы создадутся автоматически через dbContext.Database.Migrate().

**Важно по базе данных:** по умолчанию используется строка подключения к LocalDB. Она находится в файле appsettings.json.
